# 1. Mục tiêu của roadmap

Roadmap này được thiết kế để bạn học **từ nền tảng lên Humanoid Robot AI Perception** trong **4 tuần**, theo cách:

- **Mỗi đợt = 2 ngày**
- **Học phase rõ ràng, có giới hạn rõ ràng**
- **Làm Mini Project sau mỗi đợt**
- **Các bài sau cộng dồn kiến thức của các bài trước**
- **Ngay từ đầu đã có OOP, Function, Module, Loop, If/Else**
- **Python / C++ / Computer Vision đều bám đúng vai trò thực tế trong pipeline perception**

# 2. Nguyên tắc học trong 4 tuần

## Nguyên tắc 1 — Không học tràn
Mỗi đợt bạn chỉ học **đúng phần được khoanh trong roadmap**, không học lan sang phần khác.

## Nguyên tắc 2 — Học xong là làm project
Sau mỗi đợt, bạn sẽ làm **1 Mini Project** bám đúng phần phase vừa học.

## Nguyên tắc 3 — Các bài sau cộng dồn kiến thức cũ
Ví dụ:
- Bài 1 dùng kiến thức đợt 1
- Bài 2 dùng kiến thức đợt 1 + đợt 2
- Bài 3 dùng kiến thức đợt 1 + 2 + 3
- ...

## Nguyên tắc 4 — Luôn có code có cấu trúc
Ngay từ bài đầu:
- có **class**
- có **inheritance**
- có **function**
- có **loop**
- có **if / else**
- có **module Python**
- có **header/source C++**

# 7. Cách dùng roadmap này cho Mini Project

## Cách học đúng
Ở mỗi đợt:

1. Học **đúng phần phase được khoanh**
2. Không học lan sang phase sau
3. Làm **1 Mini Project** bám đúng phần vừa học
4. Giữ lại project cũ để tái sử dụng ở project mới

## Quy tắc cộng dồn
Ví dụ:
- **Bài 1** dùng kiến thức đợt 1
- **Bài 2** dùng kiến thức đợt 1 + đợt 2
- **Bài 3** dùng kiến thức đợt 1 + đợt 2 + đợt 3
- ...

## Vai trò của từng mảng trong project
- **Python** → config builder / report / helper tools
- **C++** → runtime perception module
- **CV** → đọc ảnh / xử lý ảnh / geometry / stereo / depth

# 8. Kết quả sau 4 tuần

Sau 4 tuần, nếu bạn đi hết roadmap này và làm đủ mini-project, bạn sẽ có nền tảng để bước vào các hướng như:

```text
Robot Color Object Detector
Edge / Contour Detector
Shape Detector
Feature Matching Tool
Camera Geometry Inspector
Stereo Disparity Tool
Depth Estimation Workbench
Pixel-to-Robot Coordinate Project
Humanoid Perception Mini Pipeline
```

Quan trọng hơn, bạn sẽ quen với tư duy triển khai **Humanoid Robot AI Perception** theo đúng vai trò:

```text
Python  -> chuẩn bị dữ liệu / config / report / helper tools
C++     -> perception runtime / module chạy chính
CV      -> nhìn ảnh, xử lý ảnh, rút trích thông tin, geometry, depth
```
